# **App Name**: MyCode City

## Core Features:

- Library Desk Appointment Scheduling: Users can browse available library desk slots and easily book appointments through a dedicated interface.
- Real-time Parking Occupancy Display: Displays live information on parking lot availability and occupancy within Isparta, helping users find spots efficiently.
- Municipality Announcements Feed: A centralized feed for official news, public events, and important updates from the Isparta Municipality.
- AI-Powered Announcement Summarizer Tool: Utilizes an AI tool to automatically generate concise, easy-to-read summaries of longer municipality announcements, aiding quick information digestion.
- Interactive Welcome Experience: A visually appealing 'Hoş Geldiniz' screen that introduces the web application and its core services upon initial access.
- Intuitive Main Menu Navigation: A user-friendly main menu showcasing available services with clear navigation and responsive design. Interactive elements, including buttons, will feature lightly rounded corners and subtle shadows as specified.

## Style Guidelines:

- Primary color: A sophisticated 'Isparta Gül Kurusu' (Dried Rose) represented by #8D3B4A. This deep, warm hue, drawing inspiration from Isparta's rich rose heritage, provides an elegant and authoritative presence, creating effective contrast on a light background.
- Background color: A very clean 'Kırık Beyaz' (Off-White) designated as #FDFBF9. This soft, light background enhances readability and promotes a minimalist and airy feel across the application.
- Accent color: A vibrant yet harmonious magenta-purple, #C94BA8. Derived analogously to the primary hue but with increased saturation and brightness, this color is intended for interactive elements, call-to-actions, and highlights, ensuring clear visual hierarchy and engagement.
- All text: 'Inter' (sans-serif) for its modern, objective, and highly legible qualities. This font choice supports a user-friendly and minimalist design ethos, suitable for all ages and adaptable across headlines and detailed content sections.
- Icons: Simple, modern outline-style icons will be consistently applied throughout the application to maintain visual clarity and a refined, contemporary aesthetic.
- General Layout: A clean, uncluttered, and intuitive interface that prioritizes mobile responsiveness, ensuring optimal user experience across various screen sizes. Interactive elements and buttons will feature subtly rounded corners (as specified by 'rounded-xl' in TailwindCSS context) and elegant shadows ('shadow' utility), enhancing their visual prominence and clickability.
- Subtle UI animations: Discreet transitions and micro-interactions will be implemented to create a fluid, responsive user experience that complements the application's modern and elite design, without distracting from core functionality.
